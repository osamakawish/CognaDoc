Cogna

Home

This website contains the documentation of the Cogna compiler. The details, principles, and rules on how to code in Cogna are all contained in this site, along with examples.

The following list goes over a quick summary of what each topic covers.

Introduction
The basic philosophies and elements of programming in Cogna.
Devices 1
An introduction to basic such as characters, digits, and booleans. Devices are a generalization of functions and classes.
Collections 1
Very elementary devices such as sets and maps. An intro to collections.
Strings
An introduction to strings and string methods, as well as regular expressions.
Arithmetic
Basic arithmetic operations, generalizations, and digit patterns.
Logic
Quantifiers, proofs, theorems, and a generalization of modern logic.
Collections 2
Arrays, Linked Lists, Graphs, Proofs, Functions, Relations, and other collection models.
Devices 2
Device creation, functions, relations, models, proofs, and patterns.
System
The system device, covers file reading/writing, input/output, imports, run, etc.
GUI Components
The difference components of a graphical user interface.
GUI Subdevices
Working with graphical user interfaces.
Project Design
How to implement the different philosophies of Cogna.
Appendices
The tables and short form notations of all the concepts covered.
